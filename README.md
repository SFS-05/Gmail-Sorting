AI-Powered Gmail Sorting System

Production-ready Gmail email classification and labeling system built using Chrome Extension (MV3) + FastAPI backend + Gmail API + ML-based classification.

This system is designed for real-world deployment, not demo usage. It follows strict security boundaries required for Google OAuth compliance and Chrome Web Store policies.

🚀 Overview

The system automatically classifies Gmail emails into categories such as:

AI-Work

AI-Personal

AI-Promotions

AI-Spam

It supports:

Bulk sorting of existing emails

Automatic sorting of incoming emails

Dashboard metrics in extension popup

Async background processing

Secure OAuth token lifecycle management
